# Komos Pipedream Components

This directory is shaped like a Pipedream Registry component directory for Komos.

It includes:

| Action | Purpose |
| --- | --- |
| `queue-task-run` | Queue a saved Komos task with optional inputs and an idempotency key. |
| `get-task-run` | Fetch the latest status and outputs for a Komos task run. |
